Used Open Source Libraries

Following open source projects are used on this website:

  • Zola - a rust based static site engine to generate this website!
  • Normalize.css - a good CSS reset
  • Leaflet JS - JS library to display maps
  • mpetazzoni/leaflet-gpx - plugin to display GPX files on the map
  • brunob/leaflet.fullscreen - plugin to enable fullscreen mode for the Leaflet maps
  • KaTeX - to render latex equations!
  • Ikonate - customisable & accessible vector icons, which are used on all the icons on this website!
  • Tabler.io - Icons - open source icons! Used for some link symbols
